# Software Requirements

The following software will be used throughout the course. Please ensure you have these installed and configured correctly.

## Operating System
*   **Ubuntu 20.04 LTS or 22.04 LTS:** Recommended for optimal compatibility with ROS 2 and NVIDIA Isaac SDK.
*   **Windows 10/11 with WSL2 (Windows Subsystem for Linux 2):** A viable alternative for Windows users.

## Core Robotics Frameworks
*   **ROS 2 Humble Hawksbill (or Iron Irwini):** The primary robotics middleware.
    *   Installation: [ROS 2 Documentation](https://docs.ros.org/en/humble/index.html)
*   **Gazebo Garden (or Fortress):** For 3D robot simulation.
    *   Installation: [Gazebo Documentation](https://gazebosim.org/docs)

## AI & Simulation Platforms
*   **NVIDIA Isaac SDK (latest version):** For AI-powered perception and manipulation.
    *   Installation: [NVIDIA Isaac SDK Documentation](https://docs.nvidia.com/isaac/latest/index.html)
*   **Unity (latest LTS version):** For high-fidelity visualizations and potential simulation.
    *   Installation: [Unity Download Archive](https://unity3d.com/get-unity/download/archive)

## Development Tools
*   **Python 3.8+:** Primary programming language.
*   **C++ Compiler (GCC/G++):** For ROS 2 development.
*   **Visual Studio Code:** Recommended IDE with extensions for Python, C++, and ROS 2.
*   **Git:** Version control system.

## Libraries & Dependencies
*   **CUDA Toolkit:** For NVIDIA GPU acceleration (version compatible with your GPU and Isaac SDK).
*   **cuDNN:** NVIDIA Deep Learning GPU library.
